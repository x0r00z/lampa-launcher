# Lampa Launcher

Полноэкранный веб-браузер на базе Electron для запуска Lampa с поддержкой cookies и отключенным CORS.

## Особенности

- ✅ Полноэкранный режим
- ✅ Поддержка cookies с автоматическим сохранением
- ✅ CORS отключен для всех запросов
- ✅ Настройка URL для запуска
- ✅ Сохранение настроек локально
- ✅ Кроссплатформенность (Linux/Windows)
- ✅ URL по умолчанию: http://lampa.mx

## Установка

### Требования

- Node.js (v16 или выше)
- npm

### Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка приложения

### Сборка для Linux

```bash
npm run build:linux
```

Создаст файлы в директории `dist/`:
- `.AppImage` - универсальный формат для Linux
- `.deb` - пакет для Debian/Ubuntu

### Сборка для Windows

```bash
npm run build:windows
```

Создаст установщик `.exe` в директории `dist/`

### Сборка для всех платформ

```bash
npm run build
```

### Управление

- **F11** - выход из полноэкранного режима
- **Alt+F4 / F8 / Ctrl+Q** - закрыть приложение
- **F10** - Настройки
- Для изменения URL нажмите кнопку "F10"

## Использование

1. При нажатии F10 откроется окно настроек
2. Введите URL для запуска (по умолчанию: http://lampa.mx)
3. Нажмите "Сохранить и запустить"
4. Приложение откроет указанный URL в полноэкранном режиме
5. Cookies сохраняются автоматически между сессиями

## Технические детали

### Отключение CORS

CORS отключен через:
- `webSecurity: false` в webPreferences
- `--disable-web-security` флаг командной строки

### Сохранение cookies

Cookies сохраняются через Electron Session API и автоматически загружаются при следующем запуске.

### Хранение настроек

Настройки URL сохраняются в localStorage браузера Electron.
